/* ═══════════════════════════════════════════ YOVEX AI - FINAL PRODUCTION BRIDGE ═══════════════════════════════════════════ */ // رابط الـ API الخاص بـ Space بتاعك const HF_SPACE_URL = "https://yousefjoo131-yovex-ai-train.hf.space/api/predict"; // 1. نظام الدخول (الكود: 123456) function verifyCode() { const inputs = document.querySelectorAll('.code-input'); let code = ""; inputs.forEach(input => code += input.value); if(code === "123456" || code.length === 6) { toast('🔓', 'ACCESS GRANTED', 'AI Core Synchronized.'); document.querySelector('.auth-container').style.display = 'none'; const app = document.querySelector('.app-container'); app.style.visibility = 'visible'; app.style.opacity = '1'; } else { toast('❌', 'DENIED', 'Invalid Security Code.'); } } // 2. دالة المعالجة الحقيقية (ارسال الصوت للسيرفر) async function startAIProcess(taskName) { // إنشاء زر رفع ملف مخفي const fileSelector = document.createElement('input'); fileSelector.type = 'file'; fileSelector.accept = 'audio/*'; fileSelector.onchange = async () => { const file = fileSelector.files[0]; if(!file) return; toast('⚙️', 'AI ENGINE', `Processing ${file.name}...`); // تحويل الملف لـ Blob عشان نبعته للـ API const reader = new FileReader(); reader.onload = async () => { try { const response = await fetch(HF_SPACE_URL, { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ data: [{ "name": file.name, "data": reader.result }] }) }); const result = await response.json(); if (result.data) { toast('✨', 'SUCCESS', `${taskName} Completed!`); // تشغيل الصوت اللي راجع من السيرفر const audioOut = new Audio(result.data[0].url); audioOut.play(); alert("الذكاء الاصطناعي خلص! اسمع النتيجة دلوقتي."); } } catch (err) { toast('⚠️', 'ERROR', 'Check if Space is Running.'); console.error(err); } }; reader.readAsDataURL(file); }; fileSelector.click(); } // ربط الأزرار عند تحميل الصفحة document.addEventListener('DOMContentLoaded', () => { // زر الدخول const loginBtn = document.querySelector('.btn-primary'); if(loginBtn) loginBtn.onclick = verifyCode; // أزرار الكروت (Train / Cover / Mastering) document.querySelectorAll('.card-btn').forEach(btn => { btn.onclick = (e) => { e.stopPropagation(); const title = btn.closest('.card').querySelector('h3').textContent; startAIProcess(title); }; }); });
